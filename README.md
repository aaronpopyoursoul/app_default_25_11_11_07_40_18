# Vue 3 + Element Plus 現代聊天介面

## 專案說明

本專案是一個使用 Vue 3 與 Element Plus 打造的現代化聊天介面，包含以下功能：

- 使用者與 AI 訊息顯示，區分左右對齊與不同頭像。
- 支援文字訊息與檔案上傳（圖片與文件），並可預覽檔案縮圖或檔名。
- 淺色與深色主題切換，提升使用者體驗。
- 訊息區帶有平滑過場動畫與自動滾動到底部功能，方便瀏覽歷史訊息。
- 使用 Element Plus 元件加強介面互動性與美觀。
- 具備響應式設計，適用於各種裝置。

---

## 開發環境需求

- Node.js 版本 16 以上（建議最新版 LTS）
- npm 版本 8 以上

---

## 安裝與執行

1. 複製或下載本專案至本機

2. 安裝相依套件

    npm install

3. 啟動開發伺服器

    npm run serve

4. 開啟瀏覽器並前往

    http://localhost:3000

---

## 環境設定

- 若需修改專案標題或其他環境變數，請參考 `.env.example`，複製並重新命名為 `.env`。
